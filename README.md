# 📘 Lumina AI

**Lumina AI** é uma plataforma de aprendizado com Inteligência Artificial, criada para ajudar estudantes de todos os níveis — ensino médio, vestibular e universitário — a estudar de forma gratuita, prática e eficiente. Com recursos de geração de provas, explicações detalhadas e planos de estudo personalizados, Lumina democratiza o acesso à educação de qualidade.

---

## 🌟 Features

1. **Criação de Provas Personalizadas**
   Gera exames e exercícios baseados no conteúdo que o estudante deseja praticar.

2. **Banco de Questões Inteligente**
   Analisa provas passadas do ENEM, UFSC e outros vestibulares para criar questões realistas.

3. **Explicações Detalhadas**
   Fornece respostas passo a passo, do mais simples ao avançado, facilitando a compreensão.

4. **Planos de Estudo Personalizados**
   Organiza a rotina do estudante com base nos objetivos, nível de conhecimento e tempo disponível.

5. **Acesso Gratuito e Inclusivo**
   Todos os estudantes podem usar sem precisar pagar cursinhos caros.

6. **Interface Visual e Intuitiva**
   Painel moderno com chat simulado, vídeos explicativos e elementos visuais que tornam o estudo mais agradável.

---

## 🎨 Tecnologias Usadas

- **React + Vite** – para frontend moderno e rápido
- **Tailwind CSS** – estilização responsiva e moderna
- **Framer Motion** – animações suaves e interativas
- **Lucide Icons** – ícones leves e modernos
- **JavaScript / JSX** – lógica e componentes

## 🚀 Como Rodar o Projeto Localmente

1. Clone o repositório:

```bash
git clone https://github.com/seu-usuario/lumina-ai.git
cd lumina-ai
```

2. Instale as dependências:

```bash
npm install
```

3. Rode o servidor de desenvolvimento:

```bash
npm run dev
```

4. Abra `http://localhost:5173` no seu navegador para ver o projeto funcionando.
